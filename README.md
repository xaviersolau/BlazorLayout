Don't hesitate to post issues, pull requests on the project or to fork and improve the project.

## Installation

You can checkout this Github repository or you can use the NuGet packages:

**Install using the command line from the Package Manager:**
```bash
Install-Package SoloX.BlazorLayout -version 1.0.0-alpha.2
```

**Install using the .Net CLI:**
```bash
dotnet add package SoloX.BlazorLayout --version 1.0.0-alpha.2
```

**Install editing your project file (csproj):**
```xml
<PackageReference Include="SoloX.BlazorLayout" Version="1.0.0-alpha.2" />
```
